# Kubernetes Reusable Tasks

Production-ready, reusable Ansible tasks for common Kubernetes operations.

## Available Tasks

All tasks implement DoD STIG and NIST SP 800-190 security requirements.

### deploy-application.yml - Secure Application Deployment
### backup-resources.yml - Cluster Resource Backup
### health-check.yml - Comprehensive Health Checks
### scale-deployment.yml - Deployment Scaling
### rolling-update.yml - Zero-Downtime Updates

See individual task files for detailed variable documentation.
